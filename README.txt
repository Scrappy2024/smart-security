# Smart-security.cz — Coming Soon

Tento repozitář obsahuje jednoduchou statickou stránku „ve výstavbě“ (GitHub Pages).

## Nasazení
1. Nahrajte `index.html` a `logo.png` do kořene repozitáře.
2. V Settings → Pages nastavte `Deploy from a branch`, branch `main`, folder `/`.
3. (Volitelné) Pro vlastní doménu přidejte v Settings → Pages → Custom domain `www.smart-security.cz`
   a u registrátora nastavte CNAME `www -> <username>.github.io` a A záznamy pro apex doménu (185.199.108.153, .109.153, .110.153, .111.153).
