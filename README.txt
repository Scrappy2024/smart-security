Nasazení:
1) Nahrajte vše do kořene repo (branch main) → Settings → Pages → Deploy from a branch (/(root)).
2) Custom domain: smart-security.cz → Save → Enforce HTTPS.
3) Až budete chtít stránku indexovat, upravte meta robots v index.html a/nebo smažte/změňte robots.txt.
